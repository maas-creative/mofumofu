Terminal-first CLI tool

Spec-led coding agents for traceable implementation

mofumofu is not a GUI app. It is a command-line workflow for aligning agent work to specs, traceability, and release gates.

Install from GitHub

Clone the repository, install the Python package, then verify the active spec gate.

git clone https://github.com/maas-creative/mofumofu.git
cd mofumofu
python3 -m pip install .
mofu --version
mofu status --json
